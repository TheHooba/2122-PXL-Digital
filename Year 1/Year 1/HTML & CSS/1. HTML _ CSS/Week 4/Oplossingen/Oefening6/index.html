COMPETENTIEBESCHRIJVING

Je leert de nodige vaardigheden om zelf een statische website te maken. Hiervoor werken we met de gangbare standaarden, namelijk HyperText Markup Language(HTML) en Cascading Style Sheets (CSS). In het laatste deel krijg je een introductie tot meer geavanceerde standaarden en scriptingtalen om zo je eerste dunamische pagina's te bekomen. Het is de bedoeling dat je naast het hanteren van deze vaardigheden ook een gefundeerde keuze kan maken aan de hang van het doel van de website of onderdelen van een website. Je leert ook je code te documenteren, te onderhouden en te hergebruiken. Je verweft vaardigheden om multimediatoepassingen te realiseren in verschillende contexten. Om je de nodige inzichten te geven in de manier waarop dit alles werkt, zal je rechtstreeks met de code werken door middel van een text editor.
